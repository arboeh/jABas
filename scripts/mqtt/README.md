# MQTT Scripts

This folder contains Shelly scripts that bridge BLE sensor data to Home Assistant via MQTT Auto-Discovery.

## Available Scripts

| Script                                                                             | Description                                                                                                     |
| ---------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------- |
| [`mqtt-jaalee-jht-bridge.shelly.js`](./README.mqtt-jaalee-jht-bridge.shelly.js.md) | Parses BLE advertisements from Jaalee JHT temperature/humidity sensors and publishes to Home Assistant via MQTT |

## Usage

Each script is self-contained and can be deployed independently to a Shelly device with BLE support.

See the individual README for each script for configuration and deployment instructions.
